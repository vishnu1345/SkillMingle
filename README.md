## Setting up and Running the Applications

### Express Backend

1. **Navigate to the backend directory**:

2. **Install the necessary Node.js packages**:
    ```bash
    npm install
    ```

3. **Start the Node.js application**:
    ```bash
    node app.js
    ```
